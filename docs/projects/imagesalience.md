# Image Salience Detection with Bit-Plane Slicing

## Problem

This undergraduate research project explores image saliency detection and visual information analysis through bit-plane-based image processing and computational vision methods.

## Methodology

The project investigates how visually important image regions can be identified and analyzed using saliency detection techniques, feature extraction workflows, and bit-plane slicing methods. The work also introduced me to early research practices in computer vision, image representation, and machine learning.

## Contribution

This project marked the beginning of my long-term interests in computer vision, intelligent visual analysis, and AI-driven information processing systems.
