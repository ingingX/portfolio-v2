# ValveSense-v2

## Problem

The project explores how CEEMDAN signal decomposition, Support Vector Regression (SVR), and attention-based sequence learning methods can support anomaly forecasting under noisy real-world industrial conditions through modular and reproducible AI pipelines.


## Methodology

The project investigates how human gaze information may support the evaluation and interpretation of VLM reasoning behaviors under different visual input conditions. Building upon previous work on human versus neural attention in VQA systems, the project aims to analyze whether gaze-guided visual conditions can improve model interpretability and reasoning consistency.

The current methodology focuses on multimodal evaluation workflows, human gaze data processing, gaze-conditioned visual inputs, and lightweight VLM inference experiments using Qwen-based models on datasets including VQA-MHUG and ChartGaze.
