# Visualization for Computational EEG Method

## Problem

This project explores interactive visualization workflows for computational EEG analysis using Pluto Notebook-based experimental documents and scientific computing methods.

## Methodology

The project investigates how interactive notebooks and visualization techniques can support the interpretation and exploration of EEG-related computational analysis workflows, with emphasis on reproducibility, scientific communication, and human-readable experiment structures.

## Contribution

This work strengthened my interests in scientific AI workflows, interactive computational analysis, and interpretable visualization systems for research-oriented environments.
