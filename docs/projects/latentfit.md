# LatentFit

## Problem

LatentFit explores representation learning and similarity analysis through Siamese-network-based embedding architectures for image comparison and latent space visualization.

## Methodology

The project investigates how lightweight neural networks can learn semantic similarity relationships between visual inputs and generate interpretable embedding representations for similarity-aware analysis workflows. The work also includes embedding visualization and experimental studies on latent feature distributions.

## Contribution

This project strengthened my interests in representation learning, embedding analysis, interpretable machine learning, and experimental deep learning workflows.
